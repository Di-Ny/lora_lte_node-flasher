LoRa-LTE Node v3.8.0

Outil de flashage en ligne pour les nodes LoRa/LTE — branchez le device en USB et cliquez sur le mode souhaité.

⚠️ Votre navigateur ne supporte pas l'API Web Serial. Utilisez Google Chrome, Microsoft Edge ou Opera (version desktop, pas mobile).

Mise à jour firmware données conservées

Remplace uniquement le code applicatif. La NVS (clés LoRaWAN, paramètres, ICCID, buffers de records) est préservée. Choisir ce mode pour les mises à jour de routine.

Navigateur non supporté Connexion USB requise (HTTPS)

Flash usine complet efface tout

Réinscrit bootloader + table de partitions + application et efface entièrement la flash (NVS incluse). À utiliser pour un device neuf, un sinistre logiciel, ou un changement de table de partitions.

⚠️ Les clés LoRaWAN du device seront perdues — un ré-appairage sur le serveur ChirpStack sera nécessaire.

Navigateur non supporté Connexion USB requise (HTTPS)

Avant de commencer

1. Branchez le node sur un port USB-C.
2. Sur Windows, installez le driver CP210x si le périphérique n'apparaît pas (généralement détecté automatiquement).
3. Maintenez le bouton BOOT du Heltec, appuyez sur RESET, puis relâchez BOOT pour entrer en mode flash (uniquement si le flash automatique échoue).
4. Cliquez sur le bouton du mode souhaité ci-dessus et sélectionnez le port COM du device.
