LoRa-LTE Node — Flasher en ligne

Outil de flashage pour les nodes capteurs basés sur Heltec Wireless Stick Lite V3 (ESP32-S3).

⚠️ Votre navigateur ne supporte pas l'API Web Serial. Utilisez Google Chrome, Microsoft Edge ou Opera (version desktop).

Procédure : branchez le node sur un port USB-C, choisissez votre configuration ci-dessous, puis cliquez sur le mode de flash souhaité.

Si le flash automatique échoue : maintenez BOOT, appuyez sur RESET, relâchez BOOT, puis relancez.
